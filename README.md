# Registration Form

A simple and responsive **Registration Form** built using **HTML, CSS, and JavaScript**.  
This project demonstrates basic **form validation**, **user input handling**, and **UI styling** for beginners learning front-end development.

## 🚀 Features

- User-friendly registration form
- Input fields with icons
- Required field validation
- Email format validation
- Password validation
- Error messages for incorrect input
- Disabled submit button until all fields are valid
- Clean and responsive design

## 📂 Project Structure

```
registration-form/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🧾 Form Fields

The form includes the following fields:

- **Name**
- **Email**
- **Password**

Each field is validated before submission.

## ⚙️ Technologies Used

- **HTML5** – Structure of the form
- **CSS3** – Styling and layout
- **JavaScript** – Form validation and interactivity

1. Download or clone this repository.

3. Open `index.html` in your browser.
